# SymDCT
A symbolic toolkit for analytical diffusion coefficient tensor calculations in crystals from atomic-scale kinetic data.

# What is SymDCT?
SymDCT is
- graph-based
- symmetry-based
- exact
- designed for complex crystals
- suitable for first-principles or ML interaction potential based diffusion calculations
